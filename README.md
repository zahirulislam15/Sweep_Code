Follow the procedure to run porject at your localhost

### clone this project

```
git clone [project_url]
```

### install composer

```
composer install
```

### copy .env.example file

```
cp .env.example .env
```

### generate laravel app key

```
php artisan key:generate
```

### Run Laravel Project

```
php artisan serve
```

### run npm

```
npm run dev
```

and enjoy the project
